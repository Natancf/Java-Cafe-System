# ☕ Java Café POS System

Sistema de Ponto de Venda (POS) e gestão de inventário desenvolvido em Java, utilizando o padrão MVC e interface gráfica com Java Swing. 

Este projeto aplica conceitos sólidos de Programação Orientada a Objetos, incluindo encapsulamento, herança, polimorfismo e injeção de dependências.

## 🚀 Funcionalidades
* **Painel de Venda (POS):** Processamento de pedidos, cálculo de taxas e geração de recibos.
* **Gestão de Estoque:** CRUD de produtos, alertas visuais de estoque baixo e upload de imagens.
* **Dashboards:** Relatórios diários, semanais e mensais de faturamento e itens mais vendidos.
* **Persistência:** Armazenamento local de dados estruturados em formato CSV.
